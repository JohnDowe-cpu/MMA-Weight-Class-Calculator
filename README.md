# MMA Fighter Tracker (Weight Class Calculator)

MMA Fighter Tracker is a Python command-line application that creates fighter profiles, calculates MMA weight classes, saves profiles to a text file, displays saved fighters, and searches saved fighters by name.

This project started as a simple MMA weight-class calculator and was gradually refactored into a menu-driven fighter profile tracker.

## Features

* Add fighter profiles
* Enter fighter name and weight
* Automatically calculate MMA weight class
* Handle invalid weight input with `try/except`
* Save fighter profiles to `fighters.txt`
* View saved fighter profiles
* Search saved fighters by name
* Case-insensitive fighter search
* Menu-driven command-line interface

## How to Run

Run the program from the project folder with:

```bash
python3 "Weight Class Calculator.py"
```
